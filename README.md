# Weather App (Python + PyQt5)

A simple desktop weather application built using Python and PyQt5.
It fetches real-time weather data using the OpenWeatherMap API.

## Features
- Desktop GUI using PyQt5
- Real-time weather data
- Temperature and weather description
- Error handling for invalid city and API issues

## Technologies Used
- Python
- PyQt5
- Requests
- OpenWeatherMap API

## How to Run
1. Install dependencies:
   pip install pyqt5 requests

2. Add your API key in weather_app.py

3. Run the app:
   python weather_app.py

## Note
This is a desktop GUI application and runs locally.
Replace `YOUR_API_KEY_HERE` with your own OpenWeatherMap API key.
